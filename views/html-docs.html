<address>

HTML used for contact information

It is italic.



























<area>

HTML for creating a clickable area on usually an html map element

You can combine it with a title attribute for easier to-do HTML.



























<article>

HTML for creating seperated info.

Great for forums, chat messages, etc.



























<audio>

HTML for creating audio.

Used when making audio for your site.



























<b>

HTML for creating bold text.

Bold text is typically found first in documents. (it might become deprecated)



























<base>

The default "starter" for anchors/hyperlinks.

When in the <head> element and has a href attribute, it will automatically apply the attribute value to the beginning of every <a> element!



























<basefont>

The basefont element is deprecated yet usuable. It acts like the <base> element, except that it's used for font-size, color, or font-family instead of a "link starter."





























<bdo>

HTML for changing the text direction.

With this, you can use the attribute "dir!" Then, change the value to rtl! Then, the text will go right to left!



























<big>

Deprecated HTML used for making bigger text.

You should use CSS instead.



























<blockquote>

HTML for writing text that was from another source.

You can also say the cite attribute!



























<body>

HTML used to create the front-end of your site

It is required in HTML.



























<br />

HTML used as line breaks, or just plain margin

You can use this to make a broken line in HTML



























<button>

HTML for creating a simple button

One of the most common used html elements.



























<canvas>

Used for creating HTML5 games and other things.

You can use it for creating many games only using JavaScript and one html tag!



























<center>

HTML for creating center-aligned items

Deprecated, yet used on Learn Code



























<details>

HTML for open and closing things by clicking

It's controlled by the attribute "open" or "closed"



























<dialog>

HTML for open and closing things by clicking

Pretty much the same thing, same attribute, etc.



























<div>

HTML for creating divided text. You could add backgrounds and border radiuses to make it into a drawing.





























<em>

HTML used for creating italic text.

Combined with anything will work.



























<embed>

Using this you can embed videos, sites, and even flash games!





























<font>

The font tag was used to define text color, font, and more.

You can still use this, but you are better off using CSS.



























<form>

HTML for creating a form

Combine with attributes to create the ultimate form!
